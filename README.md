# Vue Chartz
Simple and responsive SVG Chart Component for Vue.

## Clone the project

``` bash
$ git clone https://github.com/gustavoquinalha/vue-chartz.git
```

## Install dependencies
``` bash
$ npm install
```
or
``` bash
$ yarn
```

## Serve with hot reload at localhost:8080
``` bash
$ npm start
```
or 
``` bash
$ yarn start
```

## Build for production
``` bash
$ npm run build
```
or
``` bash
$ yarn build
```

## Deploy
``` bash
$ npm run deploy
```
or
``` bash
$ yarn deploy
```
